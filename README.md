# Vue-Firebase users task
App is working with vue.js and firebase

### Structure:
all api methods -> /src/services/usersServices.js
```
vuex module -> /src/store/users.js
```
pages /src/pages
```
components /src/components
```
fireBase is connected in /src/main.js
```
tests /tests/unit/ 
```
## Install
```
npm install
```
### Run
```
npm run serve
```
### Test
```
npm run test:unit
```
### Run lint
```
npm run lint
```
### Run prettier
```
npm run lint --fix
```
